一、硬件基础与微控制器

  1. ARM vs DSP区别

    • ARM：通用处理器，低功耗，高效流水线，适合多任务（如Linux系统）。
    • DSP：专用信号处理器，硬件加速数学运算（如FFT），适合音视频处理。

  2. NAND Flash vs NOR Flash

    特性 NAND Flash NOR Flash
    读取速度 慢（按块读取） 快（随机访问）
    写入/擦除 快 慢
    用途 大容量存储（eMMC） 程序存储（Bootloader）
    接口 复杂（需控制器） 直接寻址

  3. CPU内核态与用户态

    • 内核态：可执行全部指令（如清内存），权限最高。
    • 用户态：仅执行子集指令，安全性高。
      切换时机：系统调用、中断、异常。

二、C/C++语言核心

  1. 指针 vs 引用

    特性 指针 引用
    初始化 可为nullptr 必须绑定有效对象
    重绑定 支持 不支持
    内存占用 额外存储地址 无额外内存
    操作 *p解引用，&p取地址 直接操作原变量

  2. const修饰符用法

    const int a = 5;          // 常量
const int* p = &a;        // 常量指针（值不可改）
int* const p = &b;        // 指针常量（地址不可改）
void func() const;        // 成员函数不修改对象

    关键点：常量指针可重指向，指针常量不可重指向。

  3. volatile作用

    • 禁用编译器优化，强制每次从内存读取（用于多线程共享变量、硬件寄存器）。
    volatile uint32_t* reg = (uint32_t*)0x4000F000;  // 硬件寄存器操作

  4. 智能指针对比

    类型 所有权 适用场景
    unique_ptr 独占 资源独占（如文件句柄）
    shared_ptr 共享（计数） 多模块共享资源
    weak_ptr 无（观察者） 打破循环引用

三、操作系统与Linux

  1. 进程 vs 线程

    特性 进程 线程
    资源 独立内存空间 共享进程内存
    创建开销 大（需复制资源） 小
    通信开销 高（需IPC） 低（直接共享变量）
    崩溃影响 不影响其他进程 导致整个进程终止

  2. 进程状态转换

    • R（运行） → S（睡眠）：等待I/O事件
    • S → R：事件发生被唤醒
    • R → Z（僵尸）：子进程终止未回收。

  3. 死锁条件与避免
    条件：互斥、占有且等待、不可抢占、循环等待。
    方案：资源预分配、按固定顺序加锁。

四、数据结构与算法

  1. 链表 vs 数组

    操作 链表 数组
    随机访问 O(n) O(1)
    插入删除 O(1) O(n)
    内存连续性 非连续 连续

  2. 快排 vs 堆排序

    • 快排：平均O(n log n)，原地排序，不稳定。
    • 堆排序：O(n log n)，空间O(1)，适合Top K问题。

  3. 判断链表有环

    bool hasCycle(ListNode* head) {
    ListNode *slow = head, *fast = head;
    while (fast && fast->next) {
        slow = slow->next;
        fast = fast->next->next;
        if (slow == fast) return true;
    }
    return false;
}

    原理：快指针步长为2，慢指针为1，相遇即有环。

五、网络协议

  1. TCP vs UDP

    特性 TCP UDP
    连接 面向连接（三次握手） 无连接
    可靠性 可靠（重传、校验） 不可靠
    速度 慢（拥塞控制） 快（低延迟）
    应用场景 文件传输、HTTP 视频流、实时游戏

  2. I²C总线设计

    • 开漏输出：避免总线冲突，支持“线与”逻辑。
    • 上拉电阻：确保空闲时SCL/SDA为高电平。

六、嵌入式专项

  1. 交叉编译配置

    set(CMAKE_SYSTEM_NAME Linux)
set(CMAKE_C_COMPILER arm-linux-gnueabihf-gcc)  # ARM编译器
set(CMAKE_SYSROOT /opt/arm-sysroot)            # 目标设备库

    避坑：禁用动态链接减少依赖（-DBUILD_SHARED_LIBS=OFF）。

  2. 中断服务程序（ISR）规范

    • 短于10μs，无系统调用（如malloc）。
    • 共享变量用volatile修饰。

附：八股文速记口诀

  • 硬件：“ARM通用DSP快，NAND大容NOR快读”
  • C++：“指针可空引用绑，const左定右指方向”
  • OS：“进程独立线程轻，死锁四条件记分明”
  • 网络：“TCP可靠UDP快，I²C上拉开漏在”

学习建议：

  1. 理解优先：结合代码实践（如手写strcpy）深化记忆。
  2. 模块化复习：按硬件→语言→OS→算法→嵌入式顺序分块突破。
  3. 真题演练：用牛客网/LeetCode刷嵌入式题库。
