每日知识点-7

1. 信号（Signal）通信机制​

  • 通俗解释
    像给进程发微信通知——内核是“微信服务器”，信号是预设的快捷消息（如SIGINT=“紧急中断！”）。进程注册“消息处理函数”决定如何响应。
  • 技术解析 
    // 注册SIGINT信号处理函数
void handle_int(int sig) { printf("Received SIGINT!\n"); }
signal(SIGINT, handle_int); 

// 发送信号（进程内部或跨进程）
kill(pid, SIGINT);  // 向指定进程发送中断信号
    特点​​：异步通信、信息量小（仅信号编号）、无数据负载
  • 应用场景
    Nginx平滑重启：Master进程收到SIGHUP后重载配置，Worker进程完成当前请求后退出

2. 管道（Pipe）的优劣与变体​

  • 通俗解释
    父子进程间的“对讲机”：

    • 普通管道：单向通话，挂断即失效（内存中）
    • 命名管道（FIFO）：创建对讲频道，任意进程可加入（文件系统中持久化）

  • 技术解析

    # 创建命名管道
mkfifo my_pipe
# 进程A写入
echo "Hello" > my_pipe  
# 进程B读取
cat < my_pipe
    类型 通信关系 生命周期
    匿名管道 仅父子/兄弟进程 随进程结束销毁
    命名管道(FIFO) 任意进程 显式删除才销毁

  • 避坑指南
    管道默认阻塞式读写：写端未关闭时读空管道会阻塞；读端未关闭时写满管道（默认64KB）也会阻塞。

3. 共享内存（Shared Memory）高性能原理​

  • 通俗解释​
    多进程共用“云端文档”：所有进程直接编辑同一内存区域，无需复制数据（最快IPC方式），但需自备“编辑锁”（同步机制）
  • 技术解析 
    // 创建共享内存段
int shm_id = shmget(IPC_PRIVATE, size, IPC_CREAT | 0666);
// 映射到进程地址空间
char* shm_ptr = shmat(shm_id, NULL, 0);  
// 写入数据
strcpy(shm_ptr, "Shared Data");
    性能优势​​：省去内核态-用户态数据拷贝，​​吞吐量提升10倍+​​（对比管道）
  • 应用场景​
    数据库写日志（WAL）：多个写进程通过共享内存追加日志，由后台线程批量刷盘。

4. 消息队列（Message Queue）解耦逻辑​

  • 通俗解释
    进程间的“邮箱系统”：发送方投递带标签的消息到队列，接收方按需取件（支持优先级），互不阻塞。
  • 技术解析 
    // 发送结构化消息
struct msgbuf { long mtype; char mtext[100]; };
msgsnd(qid, &msg, sizeof(msg.mtext), 0); 

// 接收特定类型消息
msgrcv(qid, &msg, sizeof(msg.mtext), 1, 0); // 接收mtype=1的消息
    核心能力​​：消息持久化（内核维护）、异步通信、流量削峰
  • 应用场景
    微服务架构：订单服务投递消息到队列，库存服务异步消费，避免服务宕机导致数据丢失。

5. Socket通信的跨主机扩展性​

  • 通俗解释​
    进程的“跨国电话”：通过IP+端口定位目标进程，支持TCP（可靠连接）/UDP（高速传输）
  • 技术解析 
    // 服务端绑定监听
int sockfd = socket(AF_INET, SOCK_STREAM, 0);
bind(sockfd, (struct sockaddr*)&serv_addr, sizeof(serv_addr));
listen(sockfd, 5); 

// 客户端连接
connect(cli_sock, (struct sockaddr*)&serv_addr, sizeof(serv_addr));
    协议选择​​：
    TCP：保证有序可靠（如SSH远程登录）
    UDP：低延迟容忍丢包（如视频流传输）
  • 应用场景​
    Docker容器通信：同一主机容器用Unix Domain Socket（高性能），跨主机容器用TCP Socket。

总结

机制 生活化比喻 技术锚点 工业场景
信号 微信紧急通知 kill()/signal() Nginx热加载配置
命名管道 持久化对讲频道 mkfifo+阻塞读写控制 日志收集管道
共享内存 多人编辑云端文档 shmget()+信号量同步 数据库WAL日志缓冲
消息队列 分布式邮箱 msgsnd()/msgrcv()+消息类型过滤 微服务订单库存解耦
Socket 国际长途电话 TCP三次握手/UDP无连接 Docker跨容器通信
