​【题目1】智能指针循环引用

类型​：代码改错
​题目​：修复以下代码的内存泄漏：

class Node {  
public:  
    shared_ptr<Node> next;  
};  

int main() {  
    auto node1 = make_shared<Node>();  
    auto node2 = make_shared<Node>();  
    node1->next = node2;  
    node2->next = node1; // 循环引用！  
}

答案​：将next改为weak_ptr：

class Node {  
public:  
    weak_ptr<Node> next; // 弱引用打破循环  
};

原理​：weak_ptr不增加引用计数，避免资源无法释放

【题目2】指针运算陷阱

类型​：计算题
​题目​：int a = (int)((int*)0 + 4); 的值是多少？
​答案​：​16​（假设int占4字节）。
​解析​：

(int*)0 表示地址0（看作int数组首地址）
+4 表示偏移4个int单位（4×4=16字节）

【题目3】字节对齐

类型​：计算题
​题目​：计算以下结构体大小：

#pragma pack(4)  
struct Data {  
    char a;      // 1字节  
    double b;    // 8字节  
    int c;       // 4字节  
};

答案​：​16字节​（按4字节对齐）
​对齐规则​：

a占1字节 → 补3字节对齐
b占8字节 → 需8字节对齐，但#pragma pack(4)限制为4字节对齐
最终：1+(3)+8+4 = 16字节

逐步解析

  1. (int*)0
    0 是一个整数常量，表示内存地址 0。
    (int*)0 将 0 强制转换为 int* 类型（即指向 int 的指针），表示一个指向内存地址 0 的 int 指针。
    可以理解为：假设有一个 int 数组的首地址是 0。
  2. (int*)0 + 4
    指针的加法运算：指针 + n 表示指针向后移动 n 个该类型大小的单位。
    如果 int 占 4 字节，(int*)0 + 4 表示指针从 0 向后移动 4 个 int 的位置。
    计算公式：新地址 = 原地址 + n * sizeof(int)。
    因此：(int*)0 + 4 = 0 + 4 * 4 = 16。
    结果是 16（即地址 16，仍然是一个 int* 指针）。
  3. (int)((int*)0 + 4)
    (int) 将指针 (int*)16 强制转换为 int 类型。
    指针的值是 16，转换为 int 后就是 16。
    因此 a = 16。

【题目4】实现安全strcpy

类型​：编程题
​题目​：手写安全版strcpy，避免缓冲区溢出。
​参考答案​：

char* safe_strcpy(char* dest, size_t dest_size, const char* src) {  
    if (dest == NULL || src == NULL || dest_size == 0) return NULL;  
    size_t i = 0;  
    while (i < dest_size - 1 && src[i] != '\0') {  
        dest[i] = src[i];  
        i++;  
    }  
    dest[i] = '\0'; // 确保终止符  
    return dest;  
}

关键点​：

检查输入指针有效性
限制拷贝长度（dest_size - 1）
手动添加终止符\0

【题目5】设计线程安全单例

类型​：设计题
​题目​：用C++11实现线程安全的单例模式。
参考答案​：

class Singleton {  
public:  
    static Singleton& get() {  
        static Singleton instance; // C++11保证静态变量初始化线程安全  
        return instance;  
    }  
    Singleton(const Singleton&) = delete;  
    void operator=(const Singleton&) = delete;  
private:  
    Singleton() = default;  
};

原理​：C++11规定静态局部变量的初始化具有原子性

class LeakDetector {  
public:

        void* ptr = malloc(size);  
        logAllocation(ptr, size); // 记录分配  
        return ptr;  
    }  
    static void operator delete(void* ptr) {  
        logDeallocation(ptr);     // 记录释放  
        free(ptr);  
    }  
private:  
    static std::map<void*, size_t> allocationMap;  
};
